# helmsec version — print version information

Print the version information of helmsec, including the version number and build details.

## Usage

```
helmsec version [--json]
```

## Flags

| Flag     | Description                               |
|----------|-------------------------------------------|
| `--json` | Output version information in JSON format |

## Examples

```bash
# Print version in human-readable format
helmsec version

# Print version as JSON
helmsec version --json
```
